    Array¶

    Function Description
    AAdd Appends an element to the end of an array and returns that element.
    AEval Iterates an array, evaluating a code block against each element for its side effects and returning the array unchanged.
    AEvalA Applies a code block to each element of an array, replacing elements in place with the result.
    AFill Fills a specified range of elements in an array with a given value, modifying the original array in place.
    ALen Calculates the number of elements in a given array.
    AScan Finds the index of the first array element matching a given value or condition.
    AScanExact Searches an array for the first element that exactly matches a specified value or meets a condition.
    ArrayCalc Performs a wide range of operations—such as copying, merging, filling, sorting, and calculating statistics—on an array in a single flexible call.
    ArrayNew Creates a new array with up to three specified dimensions, each with an explicit length.
    BuildArray Splits a string into an array using a specified delimiter, optionally trimming spaces from each element.
    BuildArray2 Parses a string into a two-dimensional array using customizable line and column delimiters.
    CompArray Determines whether two arrays are structurally and value-wise equal.
    DelArray Removes an element from an array at the specified index and returns the resulting array.
    ExtractCol Extracts all values from a specified column in a two-dimensional array and returns them in a new array.
    SortArray Sorts an array in-place using numeric or custom comparison logic.

    Database¶

    SQL Queries¶

    Function Description
    GetDataSet Executes a database query and returns the results as an XML string representing the dataset.
    GetDataSetEx Executes a database query and returns its result as an XML string with flexible output options.
    GetNETDataSet Retrieves a dataset from a database using a command string and optional filters, returning XML or object output as specified.
    GetSSLDataset Executes a SQL query against a database or data source and returns the results as an SSLDataset.
    LSearch Executes a SQL search and returns a single value or a specified default if the query finds no result.
    LSelect Executes a SQL query and returns the result set as an array.
    LSelect1 Executes a parameterized SQL SELECT command and returns the result as an array of rows.
    LSelectC Executes a parameterized SQL select query and returns result rows as an array of arrays.
    RunDS Retrieves data from a specified data source using parameters and returns it in the chosen format.
    RunSQL Executes a provided SQL command against the database and returns whether the operation succeeded.
    SQLExecute Executes a SQL command against a database and returns query results or execution status.

    Transactions¶

    Function Description
    BeginLimsTransaction Starts a database transaction on a specified or default LIMS connection with optional isolation level control.
    EndLimsTransaction Closes an active LIMS transaction, saving or discarding changes based on a flag, and reports the outcome.
    GetTransactionsCount Returns the number of open database transactions for a specified or default connection.
    IsInTransaction Returns true if the specified database connection currently has an open transaction; otherwise, false.

    Connections¶

    Function Description
    GetConnectionByName Retrieves a database connection object using a specified connection name.
    GetConnectionStrings Retrieves all available database connection strings configured in the system as a two-dimensional array.
    GetDBMSName Identifies the database management system (DBMS) platform for a specified or default database connection.
    GetDBMSProviderName Returns the uppercase DBMS provider identifier for a named logical database, or raises a DAL error if the name is unknown.
    GetDefaultConnection Returns the current default database connection string used for executing database operations.
    IsDBConnected Checks if a named database connection is currently established and available for use.
    LimsSqlConnect Establishes a database connection using a registered identifier and updates the connection registry.
    LimsSqlDisconnect Closes an active database connection identified by its connection name and removes it from the internal registry.
    SetDefaultConnection Changes the active default database connection for the current session and returns the previous default connection name.
    SetSqlTimeout Adjusts the SQL command timeout for a specified database connection and returns the previous timeout value.

    SQL Utilities¶

    Function Description
    AddColDelimiters Adds database-appropriate delimiters and table-qualifies each column name in an array.
    AddNameDelimiters Wraps a string name in database-appropriate delimiters for safe SQL usage.

    GetLastSQLError Returns details of the most recent SQL database error or NIL if no error has occurred.
    GetNoLock Returns the appropriate SQL no-lock query hint string for the specified database connection or platform.
    GetRdbmsDelimiter Returns the correct identifier delimiter character for a specified RDBMS and context.
    IgnoreSqlErrors Enables or disables suppression of SQL error handling within the current session.
    LimsRecordsAffected Returns the number of records affected by the most recent database operation.
    LimsSetCounter Updates a numeric counter in a database table and inserts a new row with the result.
    PrepareArrayForIn Fills an array's missing or empty entries with type-appropriate default values in place.
    ReturnLastSQLError Retrieves the most recent SQL error that occurred during a database operation.
    RetrieveLong Retrieves large binary or text data from a database column and writes it to a file.
    SQLRemoveComments Removes all comments from an SQL statement, returning the cleaned result.
    ShowSqlErrors Alters the application-wide setting that determines whether SQL errors are visibly displayed to users.
    UpdLong Updates a large field value in a database table from the contents of a file, using search criteria for row selection.
    XmlExportSql Exports SQL query results to an XML file on disk, returning an empty string on success or an error message if failed.

    GetDataSetXMLFromArray Generates an XML string representing tabular data using provided values and field definitions.
    GetDataSetXMLFromSelect Generates an XML string from the results of a database query with customizable headers, schema, and null handling.

    Function Description
    CMonth Returns the full month name for a given date value.
    CToD Converts a date string to a date value, raising an exception if the input is NIL or returning an empty date if parsing fails.
    ClientEndOfDay Calculates the end-of-day timestamp for a given date, adjusted to the client's local timezone.
    ClientStartOfDay Calculates the local start of day for a given date, adjusting for timezone differences.
    DOW Calculates the day of the week for a given date and returns it as a numeric value.
    DOY Calculates the numeric day of the year for a given date value.
    DToC Converts a date value to a formatted string using the current date format settings.
    DToS Converts a date value to a string in the yyyymmdd format.
    DateAdd Adds a specified time interval to a date and returns the resulting date.
    DateDiff Calculates the difference between two dates and returns the result in a specified unit of time.
    DateDiffEx Calculates the difference between two dates and returns the result as a time interval object.
    DateFormat Sets the application-wide date format to a new string pattern.
    DateFromNumbers Creates a date value from individual numeric components, applying defaults where values are omitted.
    DateFromString Converts a date represented as a string into a date value using optional format and culture settings.
    DateToString Converts a date value to a string using a specified or default format.
    Day Extracts the day of the month as a number from a date value.
    Hour Extracts the hour component (0–23) from a date value; raises an error if the date is NIL.
    IsInvariantDate Checks if a date value is in an invariant (unspecified) state and returns a boolean result.
    JDay Returns the numeric day of the year (1–366) for a given date value.
    LIMSDate Returns a date as a formatted string, using a provided pattern or a default format.
    LimsGetDateFormat Returns the current global date format string used for date parsing and formatting.
    LimsTime Returns the current system time as a string in the default time format.
    MakeDateInvariant Converts a supplied date or array of dates to an invariant date format, detaching it from any local context.
    MakeDateLocal Converts a date or specific date columns within an array to local time, modifying them in place.
    Minute Extracts the minute value (0–59) from a given date.
    Month Extracts the month number (1–12) from a date value, returning 0 if the date is empty.
    NoOfDays Calculates the number of days in the month of a specified date.
    Now Returns the current date and time as a date value.
    Second Extracts the seconds value (0–59) from a given date.
    Seconds Calculates the number of seconds elapsed since midnight based on the current time.
    ServerEndOfDay Sets a date value to the final millisecond of its day using server-side time boundary rules.
    ServerStartOfDay Returns the date set to midnight at the start of its day according to server rules.
    ServerTimeZone Returns the current server's UTC offset in minutes as a number.
    StringToDate Converts a formatted date string into a date value using a specified pattern.
    Time Returns the current system time as a formatted string based on application settings.
    Today Returns the current date as a date object.
    UserTimeZone Returns the user's timezone offset in minutes from UTC, falling back to the server offset when unavailable.
    ValidateDate Validates whether a given string represents a valid date in expected formats.
    Year Returns the year component from a date value.

    Function Description

    Function Description

    DocGetCabinets Retrieves the names of all cabinets available on the connected Documentum server as an array.

    DocUpdateUser Updates a user's account details, credentials, and access configuration in the Documentum repository.

    DocGetWorkflowStatus Retrieves the current status of a workflow from the Documentum repository using its identifier.

    DocRepeatWorkitem Executes a repeat operation on a workflow workitem, optionally reassigning it to specified users.
    DocResumeWorkflow Resumes a paused or interrupted Documentum workflow identified by a string ID and returns whether the operation succeeded.
    DocStartWorkflow Initiates a new Documentum workflow process with provided workflow and document information, returning workflow context.
    DocStopWorkflow Stops an active Documentum workflow given its unique identifier.

    Function Description
