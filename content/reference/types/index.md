# Types

**8 core SSL types** — the building blocks of every SSL expression.

| Type | Description |
| --- | --- |
| [array](array.md) | Represents ordered, 1-based collections capable of storing values of any type, including nested arrays, with support for indexing, member access, and common collection operations. |
| [boolean](boolean.md) | Represents logical true/false values and supports basic logical operations and serialization. |
| [codeblock](codeblock.md) | Represents callable, parameterized expressions that can be evaluated with arguments. |
| [date](date.md) | Represents calendar-based values and supports date arithmetic, comparison, formatting, and serialization. |
| [netobject](netobject.md) | Provides dynamic access to external objects for property manipulation, method invocation, and type-safe serialization within SSL programs. |
| [number](number.md) | Represents numeric values in SSL, enabling arithmetic, bitwise, and comparison operations for calculation and decision logic. |
| [object](object.md) | Represents a dynamic collection of named properties supporting colon-syntax access, mutation, and XML serialization. |
| [string](string.md) | Represents text data in SSL, enabling concatenation, comparison, 1-based character indexing, and conversion operations. |
